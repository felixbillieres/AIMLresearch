# Practical AI — Vue d'ensemble du module

**Objectif :** Passer de la théorie à la pratique — construire, entraîner et évaluer de vrais modèles ML de A à Z.

---

## Les 3 modèles à construire

| Modèle                        | Tâche                                                                                  | Type de problème        |
| ----------------------------- | -------------------------------------------------------------------------------------- | ----------------------- |
| **Spam Classifier**           | Déterminer si un SMS est spam ou non                                                   | Classification binaire  |
| **Network Anomaly Detection** | Identifier du trafic réseau anormal ou malveillant                                     | Détection d'anomalies   |
| **Malware Classifier**        | Classer des malwares via des byteplots (représentations visuelles de données binaires) | Classification d'images |

---

## Workflow ML end-to-end

Ce module suit le pipeline complet d'un projet AI :

```
Exploration du dataset
        ↓
Préparation des données
        ↓
Entraînement du modèle
        ↓
Évaluation & tests
```

---

## Outils & environnement

**Jupyter Notebook** — Environnement interactif pour exécuter du code Python bloc par bloc. Les snippets du module sont conçus pour être copiés-collés et exécutés en séquence.

**Prérequis minimum pour entraîner les modèles :**

- 4 GB de RAM
- 4 cœurs CPU

Exécutable dans la VM fournie ou dans ton propre environnement.

---

## Ce que tu vas apprendre

- Manipuler des datasets réels (SMS, trafic réseau, binaires de malwares)
- Construire des pipelines ML complets en Python
- Évaluer la performance de modèles sur des données inédites
- Appliquer des techniques vues en théorie (classification, anomaly detection, vision) à des cas concrets de cybersécurité
